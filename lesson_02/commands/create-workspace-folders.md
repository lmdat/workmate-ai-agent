# Tạo thư mục
Tạo các thư mục sau (chỉ tạo mới nếu chưa có. Không được ghi đè thư mục đã có):

- `.opencode/`
- `.opencode/skills/`
- `.opencode/agents/`
- `input/`
- `output/`
- `scripts/`
- `tmp/`

# Tạo file .md

- Tạo file `AGENTS.md` có nội dung (UTF-8) như sau:
```
# User profile
Tên:
Chức vụ:

# Agent profile
Tên:
Chức vụ:
```

- Chỉ tạo mới nếu file `AGENTS.md` chưa có, không được ghi đè file hiện tại.
